代码托管

  • 代码仓库

    高性能远端 Git 仓库，分布式计算和存储，保护分支权限控制

  • Pages 服务

    免费静态网页托管和演示，自定义域名，支持 HTTPS

  • 代码评审

    邀请评审，对 MR +1 或授权，历史版本分屏对比，按行评论代码

  • 代码分析

    检测分析代码，提供质量报告，设置定时分析任务自动执行代码分析

Coding 的代码托管功能除了基本的 Git 代码仓库以外，还有保护分支、代码分屏对比、代码评审等高级功能，并且整合了 Pages 服务，代码分析等开发工具，提升研发效率。

WebIDE

  • 在线编码

    代码高亮,简易代码提示,实时文件保存等功能让你在浏览器中编码无忧!

  • 全功能 Terminal

    不论你需要什么样的开发环境,带 root 的全功能 Terminal 足以满足你的需求!

  • 可分享的开发环境

    你可以把配置好的开发环境一键分享给团队成员,节约团队时间.

Coding WebIDE 是 Coding 自主研发的在线集成开发环境。用户可以通过 WebIDE 创建项目的工作空间，进行在线开发，调试等操作。与此同时，WebIDE 还提供了可分享的开发环境功能，用户可以保存当前的 Terminal 环境，分享给团队的其他成员。
