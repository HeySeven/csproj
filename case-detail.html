智械科技平台

  • 平台类型

    微信移动端， PC 端

  • 时长&金额

    2 个月；25 万

  • 技术栈

    Java，MySQL，Spring，Struts，Hibernate，Frozen UI，Monarch UI

  • 地点

    杭州

  • 行业

    零售 & 电商

项目背景

IntelMachine 是基于精选推荐模式的 B2C 电商网站。 其主要运作模式是向用户提供智能机械，高科技玩具等评测，新闻资讯，视频， 并且提供给用户在线购买功能，是一个集新闻，视频，电商为一体的综合型网站。

解决方案

使用 Java + MySQL 作为主要技术方案。 使用 Spring + Struts + Hibernate 作为基础开发框架，使用 Frozen UI 驱动 H5 网页， 使用 Monarch UI 制作网站后台管理，使用阿里大于，快递 100，Ping++，七牛，微信登录等第三方服务支持， 实现快速构建涵盖下单，物流，支付等整套电商系统。

效果

项目产品规划较为完善，在快速整理完产品需求和设计后，用了约 20 天制作出了 H5 版本的网站，已经涵盖整个网站的 95% 主要功能。
